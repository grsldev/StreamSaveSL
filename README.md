# Stream Save SL

A portable GUI for yt-dlp on Windows.

## Features
- Download video and audio from 500+ sites (YouTube, Twitter, TikTok, SoundCloud...)
- Quality selection (360p, 720p, 1080p, 4K...)
- Multiple formats — video: mp4, mkv, webm / audio: mp3, opus, aac, flac, m4a, wav
- Download history
- Dark and light theme
- Portuguese and English
- Portable — no installation required

## Requirements
- Windows 10/11
- [yt-dlp](https://github.com/yt-dlp/yt-dlp) and [ffmpeg](https://github.com/BtbN/FFmpeg-Builds) inside the `bin/` folder

## Folder structure
```
StreamSaveSL/
├── StreamSaveSL.exe
└── bin/
    ├── yt-dlp.exe
    └── ffmpeg.exe
```
